Etiquetas Extras

Etiqueta pre

La etiqueta pre sirve para respetar espacios y líneas en blancp, al conrario de la etiqueta parráfo.

Hola soy Erika tengo 16 años 

Hola   soy
         Erika  tengo    16 años

Etiqueta abr

La etiqueta abr se utiliza para las abreviaturas

Según datos de la OMS el consumo excesivo de azúcar provoca daños severos en la salud de los niños

Etiqueta bdo

Si se desea cambiar la dirección de un texto, se utiliza la etiqueta bdo

Este es un texto normal

Este es un texto diferente

Etiquetas cite

La etiqueta cite se utiliza para escribir citas textuales

Que trabajen los pobres y que estudien los mensos por: Chava Iglesias

Etiquetas small

La etiqueta small hace el texto más pequeño

Este texto está dentro una etiqueta small

Etiqueta sub

Si se desea escribir un subíndice,se debe utilizar la etiqueta sub

1433

Etiquet sup

Si se desea escrbir un superíndice, se debe utilizar la etiqueta sup

143 3

Etiqueta time

Para escribir fechas y horas se debe utilizar la etiqueta time

Mañana es día6 de noviembre de 2024

Etiqueta adress

Para especificar al navegador que el texto se refiere a una dirección utiliza la etiqueta adress

Loma dorada No.100 Colonia Lomas de Rosales CP 89100 Tampico,Tamps
